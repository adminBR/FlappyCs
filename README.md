# FlappyCs

A gorgeous flappy bird clone made in dotNet

## Description

I'm learning dotNet, and this just works somehow

### Controls

* Arrow up to jump
* Enter or any key 2 times to restart

## Features

* Stunning graphics
* Responsive controls, at least 150ms of delay
